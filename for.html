For in/of

  • for/in for objects properties/methods
    • iterates over elements of object (properties and methods)
    • returns key names of property/method
    • can use as `object[key] / object.key` to access value
  • fot/of for iterable objects
    • iterates over arrays, string etc
    • returns object for each iteration
  • Break: leave a loop early
  • Continue: next iteration of a loop by passing any code below
  • Labeled Statement: Not recommended for use* define a location to "goto"
